# Valist Subgraph

This repo contains the Valist subgraph.

## Development

### Requirements

* node >= 14.17
* npm >= 6.14
* lerna

### Setup

```bash
git clone https://github.com/valist-io/valist-subgraph

cd valist-subgraph

npm run codegen

npm run build

npm run deploy:ganache
```
